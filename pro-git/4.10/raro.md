# 4.10 Git on the Server - Summary

- bare 저장소: working tree가 없는 저장소. 관습상 `.git` 접미사를 붙임
- 프로토콜 종류: local, SSH, HTTP, Git
- GitWeb: 읽기 전용 레포지토리 visualizer
- GitLab(self-hosted): 데이터베이스가 있는 깃 관련 웹 어플리케이션
