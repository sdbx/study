# A1.1 Appendix A: Git in Other Environments - Graphical Interfaces

- CLI는 강력하고 모든 기능이 있지만, 목적에 맞는 GUI를 사용하는 것도 도움이 됨

## `gitk` & `git-gui`

- `gitk`: 히스토리 뷰어. `git log` + `git grep`의 GUI 버전
- `git-gui`: 커밋 작성 도구
